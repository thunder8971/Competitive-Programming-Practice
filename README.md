# Competitive Programming Practice

This repository collects my competitive programming solutions and practice work. It is organized by platform and topic so I can revisit old problems, compare approaches, and keep improving over time.

## Goals

- Reach Candidate Master - ongoing
- Become Expert - done
- Become Cyan - done
- Become Green - done

## Repository Layout

- `BAEKJOON/` - Baekjoon Online Judge solutions
- `Codeforces/` - Codeforces problems grouped by contest or round
- `CSES/` - CSES problem set solutions organized by topic
- `Newcomers-Training/` - ICPC Assiut University Community training sheets
- `OIAJ/` - Olympiad-related practice problems
- `OnlineJudge/` - miscellaneous online judge problems
- `SPOJ/` - SPOJ solutions
- `template.cpp` - reusable competitive programming template
